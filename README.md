# Table Puzzle

Jogo 2D de lógica em que você precisa sentar 14 personagens em 7 mesas (4 cadeiras por mesa) sem colocar personagens incompatíveis na mesma mesa.

## Como jogar

1. Abra o arquivo `index.html` no navegador.
2. Arraste cada personagem para uma cadeira.
3. Observe as mensagens de conflito por mesa.
4. Vença quando todos os personagens estiverem sentados e não houver conflitos.

## Regras implementadas

- 14 personagens únicos com descrições de preferências/conflitos.
- 7 mesas com 4 cadeiras cada.
- Validação automática de conflitos em tempo real.
- Mensagem de vitória quando todos estão acomodados sem incompatibilidades.
